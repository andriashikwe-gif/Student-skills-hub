# Student Skills Hub

Student Skills Hub is a responsive educational website designed to help university students improve their academic and professional skills.

## Project Overview

This project demonstrates the use of HTML5, CSS3, JavaScript, and Bootstrap to create a responsive, multi-page educational website.

## Features

- Responsive design
- Seven interconnected web pages
- Bootstrap navigation bar
- Contact form with JavaScript validation
- Dark mode toggle
- Consistent layout and styling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Git & GitHub
